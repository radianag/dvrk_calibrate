# Install
* download & compile dvrk_calibrate

```sh
# cd to catkin ws src dir
cd /PATH/TO/CATKIN_WS/src
# clone repo
git clone https://github.com/WPI-AIM/dvrk_env.git
# build
cd ..
catkin_make
```
